Chapter 2: Registering Licenses

Before Starting Registration

Before starting the license registration process, make sure that the following information available:

  1. Registration Number for the license – this comes as an e-mail from Ixia to the Lab Manager and should have been forwarded to the person in charge of the registration.

Online License Registration – Step by Step

License registration may be initiated either:

    • When the software product that requires the license is installed. A prompt is displayed at the end of the normal software installation. The automatic license registration process can be cancelled and invoked at a later time.
    • After the software product is installed. However, it is also possible to initiate the license registration process at a later time, from any chassis or workstation that has the Ixia Registration Utility (IRU) installed. This is accomplished by running the (IRU).

Make sure the network connectivity requirements discussed in Installation Scenarios are understood, and that the necessary information discussed above in Before Starting Registration is available.

If the Installation Host is connected to the Internet (and to the lab network as well), an online registration can be performed. The steps in the online registration process are:

    • Open the Ixia Registration Utility
    • Step 1: Enter Registration Number and Password
    • Step 2: Enter a Chassis or Workstation Name
    • Step 3: Enter License Server Name
    • Step 4: Save the License

If the Installation Host is not attached to the Internet, follow the instructions in Offline License Registration – Step by Step.

Figure 2-1 will help the user keep track of the steps in the online registration process. It is repeated at each step with the current operation highlighted.

Figure 2-1. Online Registration

Note. Depending on whether the license is being installed to a computer or a chassis, the wording in the IRU dialogs is slightly different.

Open the Ixia Registration Utility

When the IRU is opened, the screen shown in Figure 2-2 is displayed. This is the main IRU screen that allows the user to select various registration operations.

To open the IRU manually, select Start > Programs > Ixia > Licensing > IRU from the start menu.

Figure 2-2. Ixia Registration Utility Main Dialog

    • Select the Register radio button.
    • Select the Start button.
Note. The other radio button operations are described in the following chapters:
  • De-registration – Chapter 3, De-registering Licenses.
  • Upgrade – Chapter 4, Updating Licenses.
  • Manage Licenses – Chapter 6, Managing Licenses.

Step 1: Enter Registration Number and Password

After selecting the Register radio button and then selecting Start, the first screen displayed is shown in Figure 2-3. If the Installation Host is not attached to the Internet, the screen shown in Figure 2-9 is displayed. Follow the instructions in Offline License Registration – Step by Step.

Figure 2-3. Registration Step 1 – Registration Number

    • Enter the Registration Number and Password exactly as specified in the Ixia generated e-mail. The source and contents of this e-mail are discussed in Before Starting Registration and in What is License Management?
    • Press the Next button.

Step 2: Enter a Chassis or Workstation Name

Note. This screen will not appear if a floating license is being installed, as floating licenses are located on a stand-alone License Servers. The process immediately jumps to Step 3: Enter License Server Name instead.

The next screen is shown in Figure 2-4. The registration number entered in the previous step internally specifies whether the license applies to an Ixia chassis or workstation. The text within the dialog will include the words Ixia Chassis or workstation as appropriate.

Figure 2-4. Registration Step 2 – Ixia Chassis or Workstation Name

    • Enter the host name or IP address of the Ixia chassis or workstation that will use the licensed software. Localhost is the default and refers to the Installation Host.
    • Look at the Product Features box and verify that the features listed correspond to the software or license being installed.
    • Press the Next button.

Step 3: Enter License Server Name

Notes. If a License Server is used, the Lab Manager should provide its name. The use of a License Server is discussed in more detail in Appendix A, License Management Planning Guide.
This step immediately follows Step 1: Enter Registration Number and Password if a floating license is being installed.

A License Server is an optional computer used to manage licenses in a lab. Figure 2-5 displays the dialog where the License Server is specified.

Figure 2-5. Registration Step 3 – License Server

    • If the lab uses a License Server, select No and enter its name in the Server name or IP address field. The license is installed on the License Server specified in the screen above.
    • If a License Server will not be used, select Yes. The license is installed on the chassis or workstation specified in Step 2: Enter a Chassis or Workstation Name.
    • Press the Next button.

Step 4: Save the License

All the necessary information has been entered. A screen similar to the one shown in Figure 2-6 is displayed.

Figure 2-6. Registration Step 4 – Save License

    • Verify the information shown.
    • Press the Next button to install the license. A license file is installed in the chassis or workstation specified in Step 2: Enter a Chassis or Workstation Name, or in Step 3: Enter License Server Name if a License Server is used.

After the license has been installed, a Receipt is displayed, similar to the one shown Figure 2-7. This is a record of the license's registration. Save the information as a file or a printout.

Figure 2-7. Registration Step 5 – Receipt

    • To save the receipt to disk, press the Save Receipt... button and designate a file.
    • To make a printout of the receipt, press the Print Receipt... button and select a printer.
    • Press Finish to exit the registration process.

The license is now available for use on the chassis or workstation. If it is a single node-locked license, it can only be used on that particular chassis or workstation.

Offline License Registration – Step by Step

The steps in the offline registration process are:

    • Step 1: Start Offline Registration
    • Step 2: Enter Registration Number and Password
    • Step 3: Enter a Chassis or Workstation Name
    • Step 4: Enter License Server Name
    • Step 5: Move to the Online Host
    • Step 6: Online – Connect to the Ixia Registration Website
    • Step 7: Online – Enter Registration Number and Password
    • Step 8: Online – Enter the Node ID
    • Step 9: Online – Save the License File
    • Step 10: Install the License

If the Installation Host is attached to the Internet, follow the instructions in Online License Registration – Step by Step.

Figure 2-8 is designed to help keep track of the steps in the offline registration process. It is repeated at each step with the current operation highlighted.

Figure 2-8. Offline Registration

Note. Depending on whether the license is being installed to a computer or a chassis, the wording in the IRU dialogs is slightly different.

Step 1: Start Offline Registration

Open the IRU as described in Open the Ixia Registration Utility.

The first screen that displayed is shown in Figure 2-9. This message indicates that the Installation Host cannot reach the Ixia website via the Internet. If the Installation Host is connected, use the following steps to diagnose the problem.

If the Installation Host should be attached to the Internet

    • Use a browser to go to any non-Ixia Internet page. If this fails, then there is a problem within the Installation Host.
    • Use the browser to access Ixia's registration server at http://licensing.ixiacom.com. If this fails, then it might be the case that Ixia's server is down. Contact Ixia at the appropriate support center to confirm.
    • If the Installation Host can access Ixia's registration server's web page, and the screen shown in Figure 2-9 still appears after restarting the license registration process, then either proceed with the steps in this section, or contact Ixia at the appropriate support center for assistance.

To proceed with offline license registration

The dialog shown in Figure 2-9 is displayed when the Installation Host is not connected to the internet.

Figure 2-9. Offline Registration Step 1 – Offline Notice

    • Find a computer that is connected to the Internet. It is necessary to transfer small amounts of data between the Online Host (the computer connected to the Internet) and the Installation Host (the chassis or workstation running the IRU). A removable USB drive or a common e-mail system are two options.
    • Press the Next button.

Step 2: Enter Registration Number and Password

The next screen displayed is shown in Figure 2-10.

Figure 2-10. Offline Registration Step 2 – Registration Number

    • Enter the Registration Number and Password exactly as specified in the Ixia generated e-mail. The source and contents of this e-mail are discussed in Before Starting Registration and in What is License Management?.
    • Press the Next button.

Step 3: Enter a Chassis or Workstation Name

Notes. This screen will not appear if a floating license is being installed, as floating licenses are located on a stand-alone License Servers. The process immediately jumps to Step 4: Enter License Server Name instead.

The next screen is shown in Figure 2-11. The registration number entered in the previous step specifies internally whether the license applies to an Ixia chassis or workstation. The text within the dialog will include the words Ixia Chassis or workstation as appropriate.

Figure 2-11. Offline Registration Step 3 – Ixia Chassis or Workstation Name

    • Enter the host name or IP address of the Ixia chassis or workstation that will be used with the licensed software. Localhost is the default and refers to the Installation Host.
    • Press the Next button.

Step 4: Enter License Server Name

Note. If a License Server is used, the Lab Manager should provide its name. The use of a License Server is discussed in more detail in Appendix A, License Management Planning Guide.
This step immediately follows Step 2: Enter a Chassis or Workstation Name if a floating license is being installed.

A License Server is an optional computer used to manage licenses in a lab. Figure 2-12 displays the dialog where the License Server is specified.

Figure 2-12. Offline Registration Step 4 – License Server

    • If a License Server is used in the Test Lab, select No and enter its name in the Server name or IP address field.
    • If no License Server is used, select Yes.
    • Press the Next button.

Step 5: Move to the Online Host

At this point, a screen similar to that shown in Figure 2-13 is displayed.

Figure 2-13. Offline Registration Step 5 – Move to Online Host

    • The information on the screen is necessary to complete the registration on the Online Host. Either:
      • Print it out using the Print... button, or
      • Save it to disk using the Save... button. This file can be copied to a removable USB drive or e-mailed to the Installation Host.
    • Move to the Online Host, start a browser and enter http://licensing.ixiacom.com.
    • Follow the instructions in:
      • Step 6: Online – Connect to the Ixia Registration Website
      • Step 7: Online – Enter Registration Number and Password
      • Step 8: Online – Enter the Node ID
      • Step 9: Online – Save the License File
    • Do not press the Next button in this dialog yet.

Step 6: Online – Connect to the Ixia Registration Website

Enter the Ixia registration web address into a browser. The first online screen displayed is shown in Figure 2-14.

Figure 2-14. Offline Registration Step 6 – Ixia Registration Website

    • Select the appropriate Ixia product. A web page similar to the one in Figure 2-15 is displayed.

      • Figure 2-15. Offline Registration Step 6 – Select Action

    • Select the Register option. A webpage similar to the one in Figure 2-16 is displayed.

Step 7: Online – Enter Registration Number and Password

Figure 2-16. Offline Registration Step 7 – Online Registration Number

    • Enter the Registration Number and Password exactly as shown in Figure 2-13.
    • Press the Go button.

Step 8: Online – Enter the Node ID

The screen shown in Figure 2-17 displays information about the license and asks the user for the Node ID. The Node ID is an internal representation of the Ixia Chassis/workstation that the license pertains to and was included as part of the information presented in Step 5: Move to the Online Host.

Figure 2-17. Offline Registration Step 8 – Enter Node ID

    • Verify the Registered To and Product/Features fields to ensure that this is the correct license to install.
    • Enter the Node ID obtained in Step 5: Move to the Online Host. In the example above, it is 01406f-414426-26e909-7400.
    • Press the Get License File button.

Step 9: Online – Save the License File

At this point, the actual Ixia license file (extension .lic) is downloaded to the Online Host. A dialog similar to that shown in Figure 2-18 is displayed.

Figure 2-18. Offline Registration Step 9 – Download License File

    • Press the Save button and select a place to save the license file.
    • Transfer this saved file over to the Installation Host via a removable USB drive or e-mail (if the Installation Host has e-mail capabilities). If a License Server is being used, then remember to save the file to the License Server (if it is a different chassis or workstation than the Installation Host). The recommended directory is C:\Program Files\Ixia\licensing\licenses.

Step 10: Install the License

Return to the Installation Host. Press the Next button. A screen similar to the one shown in Figure 2-19 is displayed.

Figure 2-19. Offline Registration Step 10 – Save License

    • Enter the filename of the license file that was copied from the Online Host as instructed in Step 9: Online – Save the License File. Use the Browse... button to help locate the file.
    • Press the Next button to install the license.

After the license has been installed, a Receipt is shown (for example, Figure 2-20). This is a record of the license's registration. Save this as a file, or make a printout of this information.

Figure 2-20. Offline Registration Step 11 – Receipt

    • To save the receipt to disk, press the Save Receipt... button and designate a file. To make a printout of the receipt, press the Print Receipt... button and select a printer.
    • Press Finish to exit the registration process.

The license is now available for use on the chassis or workstation. If it is a single node-locked license, it can only be used on that particular chassis or workstation.
